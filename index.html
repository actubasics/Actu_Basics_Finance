Step 3 - Apps Script (write from app)
1. In your Sheet: Extensions > Apps Script
2. Replace all code with the script below:
function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  const data = JSON.parse(e.postData.contents);
  if(data.action === 'add') {
    sheet.appendRow([data.date, data.etf, data.parts, data.prix]);
  } else if(data.action === 'delete') {
    const rows = sheet.getDataRange().getValues();
    for(let i = rows.length - 1; i >= 1; i--) {
      if(rows[i][0] == data.date && rows[i][1] == data.etf &&
         rows[i][2] == data.parts && rows[i][3] == data.prix) {
        sheet.deleteRow(i + 1); break;
      }
    }
  }
  return ContentService.createTextOutput(JSON.stringify({status:'ok'}))
    .setMimeType(ContentService.MimeType.JSON);
}
function doGet(e) {
  return ContentService.createTextOutput(JSON.stringify({status:'ok'}))
    .setMimeType(ContentService.MimeType.JSON);
}

Automate your investments on IBKR
Goal: buy your 4 ETFs automatically every month. IBKR offers Recurring Investments (automated purchases) and manual monthly orders. Both are described below.
Method 1 - Recurring Investments (recommended)
IBKR has a native recurring investment feature. Set an amount, frequency and day - IBKR buys automatically each month.
1
Open IBKR > Menu "Invest" > "Recurring Investments"
Available on IBKR mobile app and Client Portal (web).
2
Search the ETF by ISIN or ticker
Example: type "IE00BJ0KDQ92" or "XDWD" > select "XDWD - XETRA (EUR)". Always verify the exchange shows EUR.
3
Configure: amount, frequency, day
Amount: the monthly allocation (e.g. 320 EUR for XDWD). Frequency: Monthly. Day: 1st of the month.
4
Repeat for each of the 4 ETFs
Create 4 separate recurring orders. Cash must be available on the account - set up your bank transfer a few days before.
Note: IBKR may charge 0 commission on Recurring Investments for eligible ETFs. Check in the app for the "0 commission" icon.

Technical order settings
Order type

Market order: immediate execution at market price. Simple and fast. Slight slippage possible for less liquid ETFs.

Limit order: you set a max price. More control but risk of non-execution if the market moves up.

Recommendation: Market for XDWD, EMIM, IUSN (liquid). Limit for ZPRV (less liquid, ~75 EUR/share).
